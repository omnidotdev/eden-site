# 🌿 Eden Site

Marketing and documentation website for [Eden](https://github.com/omnidotdev/eden-cli), a Rust-powered CLI tool for developer onboarding preflight checks.

## Local Development

Install dependencies:

```sh
bun i
```

Run dev server:

```sh
bun dev
```
